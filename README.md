# hello-ssr

A minimal example of server-side rendering with [server-reason-react](https://github.com/ml-in-barcelona/server-reason-react), covering SSR, streaming with Suspense, and client-side hydration.

## Routes

| Route | Technique | Description |
|---|---|---|
| `/` | `renderToString` | Static SSR, no client JS |
| `/about` | `renderToString` | Static SSR, no client JS |
| `/stream` | `renderToStream` + Suspense | Streams HTML chunks as async data resolves |
| `/hydrate` | `renderToString` + `hydrateRoot` | SSR with client-side hydration (interactive counter) |

## Project structure

```
bin/
  main.re              Server entry point (Dream routes + rendering)
  dune
universal/
  native/
    shared/
      Document.re      HTML shell with optional <script> tag
      Counter.re       Interactive counter (compiles to both native and JS)
      HydratePage.re   Page component wrapping Counter
    dune               Native library (hello_shared_native)
  js/
    dune               Melange library (hello_shared_js) + copy_files
client/
  HydrateRoot.re       Client entry point (calls hydrateRoot)
  build.mjs            esbuild bundler script
  dune                 Melange emit config
  package.json         npm deps (react, react-dom, esbuild)
static/                Generated by esbuild (gitignored)
```

Components in `universal/native/shared/` are compiled twice: to native OCaml for the server and to JavaScript via Melange for the browser. See `universal/README.md` for details.

## Prerequisites

- [opam](https://opam.ocaml.org/) with the project's local switch
- [Node.js](https://nodejs.org/) (for esbuild and the client bundle)

## Setup

```bash
make install       # opam dependencies
make install-npm   # npm dependencies (react, react-dom, esbuild)
```

## Run

```bash
make run
```

This builds the server, bundles the client JS, and starts the server at http://localhost:8080.

## Build steps (individually)

```bash
make build    # compile server (native) + Melange JS output
make bundle   # bundle client JS with esbuild into static/
make run      # build + bundle + start server
make clean    # remove all build artifacts
```
